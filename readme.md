##Superblock for iOS9
2chまとめサイトなどの広告をすべて消せます。

##使い方
**インストールするだけでは効きません。**設定アプリから  
「Safari」  
↓  
「コンテンツブロッカー」  
↓  
「Superblock」をオン  

にしてください
